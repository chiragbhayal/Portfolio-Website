# Portfolio Website - Chirag Bhayal

A modern, responsive portfolio website showcasing web development skills and projects.

## Features

- **Responsive Design** - Works on all devices
- **Interactive Animations** - Particle.js effects and smooth scrolling
- **Dynamic Content** - Skills and projects loaded from JSON
- **Modern UI** - Clean, professional design
- **Contact Form** - Get in touch functionality
- **Social Links** - LinkedIn, GitHub, Instagram integration

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Particles.js

## Project Structure

```
├── assets/
│   ├── css/           # Stylesheets
│   ├── images/        # Images and icons
│   └── js/            # JavaScript files
├── experience/        # Experience page
├── projects/          # Projects showcase
├── index.html         # Main homepage
└── skills.json        # Skills data
```

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS

## Customization

- Update personal info in `index.html`
- Modify skills in `skills.json`
- Add projects in `projects/projects.json`
- Replace images in `assets/images/`
